This is a web page to run the L* and NL* algorithm.
Their goal is to Learn an unknown language from a Teacher thanks to memebership or equivalence queries.
Setting and launching the learner
With this program, you can choose to run L* of NL* algorithms.
Some default Teacher are proposed, but you can also create your own teacher by entering a regular expression.
Regex encode :
  • The logic or is indicated with '+'
  • Epsilon is represented by '$'
  • The Kleene's star with '*'
Once you have chosen all the settings, you can run the either step-by-step or entirely the algorithm and you can go backword and foreward the history of the operation thanks the arrow of the side of the screen (the arrow on the key-board will also work).
Automata
Automata are drawn with graphviz. You can test if a word belongs to the language of the automaton writing it in the corresponding bar and if you want you can either read the word letter by letter to see the current state of the automaton changing or directly test the appartenance of a word via the "In Automaton" button.
It is important to reset the automaton after having read a word if you want to test a new one.
